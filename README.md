# Лендинг AI
Тестовое задание на позицию HTML-верстальщика в компанию TutorPlace

[Результат](https://tutorplace-test-hpns45dz5-masha-mashas-projects.vercel.app/)

## Используемые технологии

*   **HTML5:** Для семантической разметки.
*   **SCSS:** Для модульного и поддерживаемого CSS, с применением БЭМ-методологии для именования классов.
*   **JavaScript:** Для интерактивных функций (модальное окно, мобильное меню).
*   **Gulp:** Автоматизация задач для компиляции SCSS, сборки JS и обработки файлов.
*   **Vercel:** Для деплоя и хостинга.
